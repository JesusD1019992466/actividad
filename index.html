Sobre Mí

Soy un desarrollador frontend apasionado por crear interfaces de usuario que no solo se ven increíbles, sino que funcionan perfectamente en cualquier dispositivo. Mi enfoque se centra en la experiencia del usuario y la accesibilidad web.

Con más de 3 años de experiencia, me especializo en CSS moderno, responsive design y frameworks como Bootstrap y TailwindCSS. Siempre en búsqueda de las mejores prácticas y las tecnologías más actuales para crear soluciones web innovadoras.

Creo firmemente que el código limpio y la atención al detalle hacen la diferencia entre una página web común y una experiencia digital excepcional.
